# 答题小程序

**develop是目前已上线的主分支，包含云函数版接口**

**master不再进行更新，不包含云函数，且存在数据权限问题**

#### 介绍
答题，题目解析，分享，模拟考试，排名，签到功能，获得积分

利用微信小程序方便进行答题操作

增加了汉字查询、成语查询、英汉词典等功能

交流 qq群：123955944

gitee地址 [https://gitee.com/siival/question](https://gitee.com/siival/question)
